# voting-system-c
simple voting system in C language

## Features
- User-defined number of candidates
- Each voter can vote only once
- Invalid votes are ignored
- Vote counting using one-dimensional array
- Handles tie situations (multiple winners)
- Compatible with Dev-C++

## How to Run
1. Compile using GCC or Dev-C++
2. Run the program
3. Enter number of candidates and voters
4. Input votes as candidate numbers
